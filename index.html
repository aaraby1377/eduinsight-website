Eduinsight Smarter Tools, Better Grades

Eduinsight uses AI to analyze past papers and lecture content, uncover key patterns, and help students focus on what truly matters for better academic performance.

🌱About EduInsight

EduInsight is an AI-powered learning platform built specifically for SLIIT students regarding DMS module. Developed by final-year undergraduates at SLIIT, it integrates cutting edge technologies such as:

🤖

AI-Based Content Analysis

Develop an intelligent system to analyze past exam papers and lecture materials in order to generate model exam questions and concise, exam-focused notes for SLIIT students.

• Feature: Model Question Generation

📚

Syllabus-Aware Learning Support

Design a system that delivers structured learning resources including summaries, flashcards, audio explanations, and continuous assessment (CA) guidance, aligned with the SLIIT curriculum.

• Feature: Structured Learning

📊

Adaptive Evaluation & Feedback System

Evaluates student answers using LLM-based semantic analysis and classifies question difficulty using Bloom's Taxonomy. Generates personalized feedback and adapts learning by identifying weak topics for targeted practice.

• Feature: Personalized Feedback & Adaptive Learning

🎯

NLP-Based Personalization

Implement an NLP-driven system to classify academic content and generate personalized MCQ-based study plans, tailored to individual student performance and learning needs.

• Feature: Adaptive Study Plans

🔍Research Domain

✨Key Features Overview

1

Past Paper Mining

Analyzes past exam papers and lecture slides using TF-IDF, keyphrase extraction, and semantic clustering to generate model exam papers and aligned short notes for targeted exam preparation.

2

NLP-Driven Study Material Generation

A syllabus-aware learning support system that provides summaries, flashcards, and CA guidance using RAFT + RLHF pipeline to deliver syllabus-aligned notes and audio explanations.

3

AI-Driven Essay Evaluation

Predicts question difficulty and evaluates essay-type and structured responses using semantic similarity and AI-powered feedback generation, providing personalized educator insights and performance analysis.

4

Personalized Study Planner

An NLP-based personalization system that classifies educational content using MCQ classification, syllabus analysis, and topic importance mapping to generate adaptive, individualized study plans.
